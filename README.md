# همیار فرش | HamyarFarsh

> سیستم جامع مدیریت و فروش فرش ماشینی

## درباره پروژه
همیار فرش یک پلتفرم مدیریت و فروش آنلاین فرش ماشینی است که به فروشندگان فرش امکان مدیریت محصولات، قیمت‌گذاری هوشمند، سفارش‌گیری و ارتباط با مشتریان از طریق وب‌سایت و پیام‌رسان‌ها را می‌دهد.

## تکنولوژی‌ها
- **Backend:** Django 5 + MySQL (MariaDB 10.4)
- **Frontend:** Bootstrap 5 RTL (لوکال) + Vazirmatn Font (لوکال)
- **Rich Editor:** CKEditor 5 (django-ckeditor-5 — MIT License)
- **Image Processing:** Pillow + arabic_reshaper + python-bidi
- **Video Processing:** FFmpeg + ffmpeg-python
- **Calendar:** jdatetime (شمسی)
- **Charts:** Chart.js (لوکال)
- **Autocomplete:** Select2 (CDN)
- **SEO:** JSON-LD Schema.org + Open Graph + Canonical URLs
