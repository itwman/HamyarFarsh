## مشخصات فنی

| مشخصه | مقدار |
|-------|-------|
| فریم‌ورک بک‌اند | Django 5 + MySQL (MariaDB 10.4) |
| فرانت‌اند | Bootstrap 5 RTL + Vazirmatn (لوکال) |
| پردازش تصویر | Pillow + arabic_reshaper + python-bidi |
| تقویم | شمسی (jdatetime) |
| نمودارها | Chart.js (لوکال) |
| زبان | فارسی (RTL) + قالب قرمز دیجی‌کالایی |
| SEO | JSON-LD Schema.org + Open Graph + Canonical URLs |

## فرمول‌های کلیدی

### قیمت فروش 12 متری
```
قیمت_فروش = CEIL((قیمت_خرید × 1.15) + هزینه_حمل, 100000)
```

### قیمت سایر سایزها (غیر از 9 متری)
```
قیمت_سایز = (قیمت_فروش_12متری ÷ 12) × مساحت_سایز
```

### قیمت 9 متری (3.50×2.50)
```
# پرتی ثابت:
قیمت_9متری = (قیمت_فروش_12متری ÷ 12 × 9) + عدد_پرتی

# پرتی درصدی:
قیمت_9متری = (قیمت_فروش_12متری ÷ 12 × 9) × (1 + درصد_پرتی)
```

### محاسبه اقساط
```
مانده = جمع_فاکتور - پیش‌پرداخت
سود_کل = درصد_سود_ماهانه × تعداد_ماه
مبلغ_با_سود = مانده × (1 + سود_کل)
مبلغ_هر_قسط = مبلغ_با_سود ÷ تعداد_اقساط
```

**مثال:** فاکتور 50M, پیش‌پرداخت 20M, 12 چک ماهانه 2.5%:
- مانده = 30M
- سود = 2.5% × 12 = 30%
- مبلغ = 30M × 1.30 = 39M
- هر چک = 39M ÷ 12 = 3,250,000 تومان
